# 3. Key Features (주요 기능)

* **지출 내역 추가**

  * 사용한 용돈의 금액을 입력할 수 있습니다.
  * 식비, 문구, 교통, 여가, 저축, 기타 중 하나의 카테고리를 선택할 수 있습니다.
  * 지출 내용을 메모로 함께 저장할 수 있습니다.

* **수입 내역 추가**

  * 받은 용돈이나 추가 수입 금액을 입력할 수 있습니다.
  * 수입 내용을 메모로 함께 저장할 수 있습니다.

* **전체 내역 조회**

  * 지금까지 입력한 수입과 지출 내역을 한 번에 확인할 수 있습니다.
  * 구분, 카테고리, 금액, 메모가 함께 출력됩니다.
  * 전체 수입과 지출을 계산하여 총 잔액을 확인할 수 있습니다.

* **카테고리별 조회**

  * 식비, 문구, 교통, 여가, 저축, 기타 카테고리별 지출 금액을 확인할 수 있습니다.
  * 어떤 항목에 용돈을 많이 사용했는지 쉽게 파악할 수 있습니다.

* **프로그램 종료**

  * 메뉴에서 0번을 선택하면 프로그램을 종료할 수 있습니다.

# 7. Function Structure (함수 구조)

|          함수명         | 기능                              |
| :------------------: | ------------------------------- |
|       `main()`       | 메인 화면을 출력하고 사용자의 메뉴 선택을 처리합니다.  |
|    `add_expense()`   | 지출 금액, 카테고리, 지출 내용을 입력받아 저장합니다. |
